## Самостоятельная работа №1
### Ресторан на предприятии ведет учет посещений за неделю при помощи
кода работника. У них есть список со всеми посещениями за неделю.
Ваша задача почитать:
• Сколько было выдано чеков
• Сколько разных людей посетило ресторан
• Какой работник посетил ресторан больше всех раз
Список выданных чеков за неделю:
[8734, 2345, 8201, 6621, 9999, 1234, 5678, 8201, 8888, 4321, 3365,
1478, 9865, 5555, 7777, 9998, 1111, 2222, 3333, 4444, 5556, 6666,
5410, 7778, 8889, 4445, 1439, 9604, 8201, 3365, 7502, 3016, 4928,
5837, 8201, 2643, 5017, 9682, 8530, 3250, 7193, 9051, 4506, 1987,
3365, 5410, 7168, 7777, 9865, 5678, 8201, 4445, 3016, 4506, 4506]
Результатом выполнения задачи будет: листинг кода, и вывод в
консоль, в котором будет указана вся необходимая информация.

```python
from collections import Counter

checks = [8734, 2345, 8201, 6621, 9999, 1234, 5678, 8201, 8888, 4321, 3365,
          1478, 9865, 5555, 7777, 9998, 1111, 2222, 3333, 4444, 5556, 6666,
          5410, 7778, 8889, 4445, 1439, 9604, 8201, 3365, 7502, 3016, 4928,
          5837, 8201, 2643, 5017, 9682, 8530, 3250, 7193, 9051, 4506, 1987,
          3365, 5410, 7168, 7777, 9865, 5678, 8201, 4445, 3016, 4506, 4506]

num_checks = len(checks)
num_unique_visitors = len(set(checks))
most_frequent_visitor = Counter(checks).most_common(1)[0][0]

print(f"Количество выданных чеков за неделю: {num_checks}")
print(f"Количество разных людей посетивших ресторан: {num_unique_visitors}")
print(f"Работник с кодом {most_frequent_visitor} посетил ресторан больше всех раз.")
```

## Самостоятельная работа №3
### Преподаватель по математике придумал странную задачку. У вас есть
три списка с элементами, каждый элемент которых – длина стороны
треугольника, ваша задача найти площади двух треугольников,
составленные из максимальных и минимальных элементов полученных
списков. Результатом выполнения задачи будет: листинг кода, и вывод
в консоль, в котором будут указаны два этих значения.
Три списка:
one = [12, 25, 3, 48, 71]
two = [5, 18, 40, 62, 98]
three = [4, 21, 37, 56, 84]

```python
def triangle(a, b, c):
    s = (a + b + c) / 2
    area = (s * (s - a) * (s - b) * (s - c)) ** 0.5
    return area

one = [12, 25, 3, 48, 71]
two = [5, 18, 40, 62, 98]
three = [4, 21, 37, 56, 84]

max_one = max(one)
min_one = min(one)

max_two = max(two)
min_two = min(two)

max_three = max(three)
min_three = min(three)

area_max = triangle(max_one, max_two, max_three)
area_min = triangle(min_one, min_two, min_three)

print("Площадь треугольника из максимальных значений:", area_max)
print("Площадь треугольника из минимальных значений:", area_min)
```

[2, 3, 4, 5, 3, 4, 5, 2, 2, 5, 3, 4, 3, 5, 4]
[4, 2, 3, 5, 3, 5, 4, 2, 2, 5, 4, 3, 5, 3, 4]

Результатом выполнения задачи будет: листинг кода, и вывод в

```python
def fix_grades(grades):
    i = 0
    while i < len(grades):
        if grades[i] == 2:
            del grades[i]
        elif grades[i] == 3:
            grades[i] = 4
        else:
            i += 1

grades1 = [2, 3, 4, 5, 3, 4, 5, 2, 2, 5, 3, 4, 3, 5, 4]
grades2 = [4, 2, 3, 5, 3, 5, 4, 2, 2, 5, 4, 3, 5, 3, 4]
grades3 = [5, 4, 3, 3, 4, 3, 3, 5, 5, 3, 3, 3, 3, 4, 4]

fix_grades(grades1)
fix_grades(grades2)
fix_grades(grades3)

print("Обновленный массив 1:", grades1)
print("Обновленный массив 2:", grades2)
print("Обновленный массив 3:", grades3)
```

## Самостоятельная работа №5
### Вам предоставлены списки натуральных чисел, из них необходимо
сформировать множества. При этом следует соблюдать это правило:
если какое-либо число повторяется, то преобразовать его в строку по
следующему образцу: например, если число 4 повторяется 3 раза, то в
множестве будет следующая запись: само число 4, строка «44», строка
«444».
Множества для теста:
list_1 = [1, 1, 3, 3, 1]
list_2 = [5, 5, 5, 5, 5, 5, 5]
list_3 = [2, 2, 1, 2, 2, 5, 6, 7, 1, 3, 2, 2]
Результаты вывода (порядок может отличаться, поскольку мы работаем
с set()):
{'11', 1, 3, '33', '111'}
{5, '5555', '555555', '55555', '555', '55', '5555555'}
{'11', 1, 3, 2, 5, 6, '222222', '222', 7, '2222', '22222', '22'}

```python
def generate_sets(numbers):
    result_set = set()

    for number in numbers:
        count = numbers.count(number)
        if count > 1:
            result_set.update(str(number) * i for i in range(1, count + 1))
        else:
            result_set.add(number)

    return result_set

list_1 = [1, 1, 3, 3, 1]
list_2 = [5, 5, 5, 5, 5, 5, 5]
list_3 = [2, 2, 1, 2, 2, 5, 6, 7, 1, 3, 2, 2]

set_1 = generate_sets(list_1)
set_2 = generate_sets(list_2)
set_3 = generate_sets(list_3)

print("Результаты вывода для list_1:", set_1)
print("Результаты вывода для list_2:", set_2)
print("Результаты вывода для list_3:", set_3)
```
