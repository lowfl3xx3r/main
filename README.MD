декоратор.
Подсказка: необходимо использовать модуль time
Декоратор необходимо использовать для этой функции:
def fibonacci():
fib1 = fib2 = 1
for i in range(2, 200):
fib1, fib2 = fib2, fib1 + fib2
print(fib2, end=' ')
if __name__ == '__main__':
fibonacci()

```python
import time
def calculate_time(func):
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        print(f"\nВремя на выполнение {func.__name__}: {end_time - start_time} секунд")
        return result
    return wrapper
@calculate_time
def fibonacci():
    fib1 = fib2 = 1
    for i in range(2, 200):
        fib1, fib2 = fib2, fib1 + fib2
        print(fib2, end=' ')
if __name__ == '__main__':
    fibonacci()
```

## Самостоятельная работа №4
### Создать собственный декоратор.
Создадим собственный декоратор, который будет выводить название функции и ее аргументы перед выполнением функции. После выполнения функции он также выведет сообщение о завершении ее работы.
Создается класс декоратора CustomDecorator.
В конструкторе класса сохраняется декорируемая функция.
Переопределяется метод __call__, который вызывается при вызове объекта декоратора.
В методе __call__ выводится информация о том, какая функция вызывается и с какими аргументами, а затем вызывается декорируемая функция.
После выполнения функции выводится сообщение о ее завершении.
Функции add и multiply декорируются этим классом, что позволяет использовать его функциональность для этих функций.

```python
# Создаем класс декоратора
class Logger:
    def __init__(self, func):
        self.func = func  # Сохраняем декорируемую функцию
    def __call__(self, *args, **kwargs):
        print(f"Вызывается функция: {self.func.__name__}")
        result = self.func(*args, **kwargs)  # Вызываем декорируемую функцию
        return result
# Примеры функций, которые будут использоваться с декоратором
@Logger
def greet(name):
    return f"Привет, {name}!"
@Logger
def square(n):
    return n ** 2
# Тесты
result_greet = greet("Виктор")  # Выводится информация о вызове функции greet
print("Результат:", result_greet)  # Выводится результат выполнения функции greet
result_square = square(5)  # Выводится информация о вызове функции square
print("Результат:", result_square)  # Выводится результат выполнения функции square
```

## Самостоятельная работа №5
### Создайте собственное исключение, которое будет использоваться в
двух любых фрагментах кода. Исключения, которые использовались
ранее в работе нельзя воссоздавать. Результатом выполнения задачи
будет: класс исключения, код к котором в двух местах используется это
исключение, скриншот консоли с выполненной программой и
подробные комментарии, которые будут описывать работу вашего
кода.

```python
# Создаем класс собственного исключения
class DataProcessingError(Exception):
    def __init__(self, message="Ошибка при обработке данных"):
        self.message = message
        super().__init__(self.message)
# Пример использования собственного исключения
def process_data(data):
    try:
        if not data:
            raise DataProcessingError("Отсутствуют данные для обработки")
        else:
            print("Данные обработаны успешно")
    except DataProcessingError as er:
        print("Ошибка:", er)
def calculate_average(numbers):
    try:
        if not numbers:
            raise DataProcessingError("Отсутствуют числа для вычисления среднего значения")
        else:
            average = sum(numbers) / len(numbers)
            print("Среднее значение:", average)
    except DataProcessingError as er:
        print("Ошибка:", er)
# Тесты
try:
    process_data([])  # Вызывается исключение из process_data
except DataProcessingError as er:
    print("Ошибка:", er)
try:
    calculate_average([])  # Вызывается исключение из calculate_average
except DataProcessingError as er:
    print("Ошибка:", er)
```
